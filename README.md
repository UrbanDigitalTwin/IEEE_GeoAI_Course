# GeoAI for Disaster Management

This repository contains a basic 10-week introductory course on Geospatial Artificial Intelligence (GeoAI) for disaster management applications.

The course introduces learners to geospatial data, flood-risk concepts, mapping, machine learning, image-based analysis, decision-support workflows, and responsible AI use in disaster contexts.

## Course Level

Beginner to early intermediate.

This course is designed for students or professionals who are new to GeoAI and want a structured introduction using practical examples.

## Course Duration

10 weeks

## Course Structure

| Week | Topic |
|---|---|
| Week 0 | Course orientation and setup |
| Week 1 | Introduction to GeoAI and disaster management |
| Week 2 | Geospatial data basics |
| Week 3 | Remote sensing and imagery |
| Week 4 | Machine learning foundations |
| Week 5 | Object detection and segmentation concepts |
| Week 6 | Flood-risk mapping and severity assessment |
| Week 7 | Urban infrastructure and accessibility analysis |
| Week 8 | Decision-support workflows |
| Week 9 | Responsible GeoAI and limitations |
| Week 10 | Final integration and reflection |

## Learning Objectives

By the end of this course, learners should be able to:

1. Explain the role of GeoAI in disaster management.
2. Identify common geospatial data types used in disaster applications.
3. Interpret basic flood-risk and severity-mapping outputs.
4. Describe how AI models can support object detection, segmentation, and situational awareness.
5. Recognize limitations, uncertainty, and ethical concerns in GeoAI-based decision support.

## How to Use This Course

1. Start with the course landing page.
2. Complete each weekly module in order.
3. Open the notebooks in Google Colab when links are provided.
4. Complete the quizzes and reflection activities.
5. Use the final week to connect the concepts into a complete disaster-management workflow.

## Important Note

This course is intended for educational purposes. The examples and notebooks are simplified demonstrations and should not be treated as operational flood-warning, emergency-response, or public-safety systems.

## Recommended Background

No advanced programming background is required, but basic familiarity with Python, maps, and data analysis will be helpful.
