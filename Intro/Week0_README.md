This folder contains the onboarding materials for the course.

## 🎯 Learning Goals
- understand how the course is organized
- verify access to notebooks and folders
- understand how to run notebooks in Colab or locally
- prepare for the weekly notebook workflow

## 🧩 Main Topics
- repository structure
- how to use GitHub for the course
- opening notebooks in Google Colab
- basic folder and output expectations
- how weekly assignments are organized

## ✅ Expected Outputs
- working course access in Colab or Jupyter
- clear understanding of folder structure

## 🧠 Weekly Workflow
1. Read the repository overview and understand the course structure
2. Open a notebook in Google Colab or a local Jupyter environment
3. Check that the required folders and outputs can be created
4. Understand how weekly notebooks should be read, run, and modified
