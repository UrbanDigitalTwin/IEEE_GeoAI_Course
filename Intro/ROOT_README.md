## 📘 Course Overview
This repository contains a **10-week GitHub-based notebook course** designed to help students learn practical **GeoAI workflows for disaster management** using Python.

The course emphasizes:
- weekly notebook-based learning
- reproducible workflows in Google Colab
- flood intelligence from geospatial and street-level data
- model evaluation and operational interpretation
- decision support and digital twin integration
- a final portfolio-style project

Students can **read, run, interpret, and modify** each notebook as they progress through the course.

---

## 🌍 What You Will Learn
- Python notebook workflows for GeoAI
- Disaster-related data sources such as SAR, optical imagery, DEMs, and hydrological data

- Surface water detection with semantic segmentation
- Vehicle-based urban flood detection from street-level scenes

## 🛠 Installation & Setup

### Option 1: Run in Google Colab
This is the recommended option for students.

Benefits:
- no local installation required
- simple notebook-based workflow
- easy sharing and submission
- suitable for classroom and self-paced learning

### Option 2: Run Locally

```bash
git clone https://github.com/your-username/geoai-disaster-course.git
cd geoai-disaster-course
pip install numpy pandas matplotlib notebook
```

Optional packages for expanded workflows:

```bash
pip install geoai-py ultralytics opencv-python
```

## 📅 Weekly Structure

### Week 0 — Course Orientation and Repository Setup
Repository setup, Colab workflow, folder structure, and course expectations.

### Week 1 — Course Setup and Introduction to GeoAI
Course onboarding, GeoAI basics, and disaster-management context.

### Week 2 — Disaster Data Sources and Visualization
SAR, optical imagery, UAVs, DEMs, rainfall, and hydrological data.

### Week 3 — ML and CV Fundamentals for Disaster Management
Classification, detection, segmentation, and evaluation metrics.

### Week 4 — Surface Water Detection with Segmentation
Segmentation workflow for detecting water from geospatial imagery.

### Week 5 — Vehicle-Based Urban Flood Detection
Vehicle detection and context-aware cropping from street-level flood scenes.

### Week 6 — Urban Flood Depth Estimation Pipeline
Multi-stage flood-depth estimation with crop-level prediction and aggregation.

### Week 7 — Model Evaluation and Error Analysis
Performance analysis, threshold tuning, and failure case interpretation.

### Week 8 — Disaster Response Decision Support
Risk categories, priority scoring, actions, and review flags.

### Week 9 — Urban Digital Twin Integration
Spatial context, infrastructure awareness, temporal updates, and city-scale interpretation.

### Week 10 — Final Project Template
Complete GitHub-ready mini project template.
