# Week 2: Disaster Data Sources and Visualization

This folder contains the notebook and supporting material for this week of the course.

## 📘 Weekly Overview
This week introduces the major geospatial and environmental data sources used in disaster-management workflows and helps students compare their roles, strengths, and limitations.

## 🎯 Learning Goals
- identify major disaster-relevant data sources
- understand the role of each data type

- optical imagery
- UAV imagery
- DEMs
- rainfall data
- river and hydrological data
