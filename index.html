About the Course

Designed for learning, research, and real-world disaster intelligence.

The course moves from core geospatial concepts to deployable AI workflows that support hazard detection, impact analysis, and decision-ready visualization.

Learning Outcomes

By the end of this course, learners will be able to:

  • Geospatial Disaster Data Foundations: Identify, access, preprocess, and visualize geospatial datasets used in disaster management, including raster data, vector layers, satellite imagery, DEMs, and open-source disaster datasets.
  • GeoAI and Computer Vision Application: Apply machine learning, computer vision, and segmentation methods to disaster-related geospatial problems such as flood detection, surface-water mapping, and urban damage assessment.
  • Multimodal Flood Intelligence Workflows: Integrate multiple data sources, including remote sensing, terrain data, street-level imagery, and contextual urban information, to support flood detection and flood-depth estimation workflows.
  • Model Evaluation and Decision Support: Evaluate GeoAI model outputs, interpret uncertainty and failure cases, and translate results into decision-support products for disaster response and planning.
  • Digital Twin Integration and Project Synthesis: Connect GeoAI outputs with urban digital twin concepts and design an end-to-end, reproducible disaster analytics project using open-source workflows.
Curriculum

Weekly notebook pathway

Each week is structured as a self-contained notebook with learning objectives, guided examples, exercises, and extension activities.

W01

Introduction to GeoAI

Course overview, Python setup, and geospatial data fundamentals.

Read more

Set up the learning environment, review the role of GeoAI in disaster management, and practice working with core geospatial data types.

W02

Disaster Data Sources & Visualization

Explore open disaster datasets, geospatial visualization, and data access workflows.

Read more

Learn where disaster, weather, remote sensing, and map data can be found, then build practical visual summaries for analysis.

W03

ML & CV Fundamentals

Core machine learning and computer vision concepts for disaster-management tasks.

Read more

Review supervised learning, image features, classification, object detection, and segmentation concepts used throughout the course notebooks.

W04

Surface Water Segmentation

Segmentation workflows for detecting surface water and flood extent.

Read more

Apply image segmentation methods to identify water surfaces, compare results, and connect mapped water extent to flood-analysis needs.

W05

Vehicle-Based Flood Detection

Computer vision workflows for identifying urban flooding from vehicle-based imagery.

Read more

Use street-level or vehicle-based imagery to recognize visible flood conditions and reason about urban scene interpretation.

W06

Urban Flood Depth Estimation

Pipeline design for estimating flood depth in urban environments.

Read more

Combine model outputs, elevation data, and urban context to understand how flood-depth estimation pipelines are assembled.

W07

Model Evaluation & Error Analysis

Evaluate model performance, inspect errors, and improve disaster AI workflows.

Read more

Measure performance with appropriate metrics, inspect failure cases, and document uncertainty in disaster-management model outputs.

W08

Disaster Response Decision Support

Translate model outputs into decision-support products for response workflows.

Read more

Design outputs that help responders interpret risk, prioritize actions, and communicate geospatial AI findings clearly.

W09

Urban Digital Twin Integration

Connect GeoAI outputs with urban digital twin concepts and systems.

Read more

Explore how disaster intelligence can feed urban digital twin workflows for monitoring, simulation, and planning.

W10

Final Project Template

Complete a final project using the course workflow template.

Read more

Use the final template to frame a problem, document data and methods, evaluate results, and present a complete GeoAI workflow.
